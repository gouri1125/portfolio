Django Portfolio Website

This is a personal portfolio website developed using Django to showcase my projects, skills, and professional profile. The website includes a clean user interface, responsive design, and an admin dashboard for easy content management. It demonstrates core Django concepts such as models, views, templates, static file handling, authentication, and deployment best practices.

The project is deployed on Render using Gunicorn and Whitenoise, with environment-based configuration for security and scalability. This portfolio is designed to be recruiter-friendly and production-ready.
